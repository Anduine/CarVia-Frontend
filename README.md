# 🚗 CarVia Frontend

Frontend частина веб-застосунку для купівлі та продажу автомобілів.

## 🛠 Технології

- React
- React Router
- Axios
- CSS (адаптивна верстка)

## ⚙️ Функціонал

- Перегляд списку лотів
- Фільтрація (ціна, рік, бренд, модель)
- Перегляд деталей лота
- Реєстрація та авторизація
- Лайки (список обраних лотів)
- Створення / редагування / видалення лотів
- Завантаження зображень
- Редагування профілю та аватара

## 📁 Структура

- `components/`
  - `ui/` - UI компоненти
  - `layout/` - хедери та футери
  - `pages/` - сторінки
- `services/` - запити до бекенду
- `providers/` - кастомні провайдери контексту
- `assets/` - стилі, та іконки
- `utils/` - службові функції

## 🔌 API

Всі запити проходять через API Gateway.

## 🚀 Запуск

```bash
npm install
npm start
```
